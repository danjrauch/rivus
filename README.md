### Prerequisites

* [Nim](https://nim-lang.org)

### Installation

## Development
1. Clone the Repo
   ```sh
   git clone https://github.com/danjrauch/rivus.git
   ```
2. Run the benchmarks
   ```sh
   nim c -o:./bin/program -d:release -r src/benchmark.nim && rm -R ./bin
   ```
